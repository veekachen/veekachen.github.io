申请github及创建静态页面

  1. 去github申请一个账号
  2. 创建一个github项目，项目名称username.github.io，username替换为自己的用户名
  3. 进入此项目，点击页面上的Setting，进入设置页面，点击Automatic page generator，Github将会自动替你创建出一个gh-pages的页面。如果你的配置没有问题，那么大约15分钟之后，username.github.io这个网址就可以正常访问了。
